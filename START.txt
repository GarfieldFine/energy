智维衡碳 — 建筑能源智能管理系统 · 快速启动
==========================================

【一、数据准备】
  与 building_energy_system 同级放置：
  · bdg_cleaned_output\          能耗 / 元数据 / 字典 CSV
  · kb_documents\ 或 知识库所需文档\   规范 PDF（索引后用于问答）
  · sft_merged\sikong_sft_all.jsonl   司空语料（可选）

【二、启动】
  1) 后端（默认 8765）
     cd building_energy_system\backend
     pip install -r requirements.txt
     python -m uvicorn app.main:app --reload --host 127.0.0.1 --port 8765
     文档 http://127.0.0.1:8765/docs

  2) 前端
     cd building_energy_system\frontend
     npm install
     npm run dev
     浏览器打开终端地址（通常 http://127.0.0.1:5173）

  3) 知识库索引（PDF 变更后执行一次）
     cd building_energy_system\backend
     python scripts\ingest_kb.py

【三、演示账号】
  admin / admin123     全部菜单 + 平台运维
  energy / energy123   能源分析、工单只读
  ops / ops123         问答、工单、孪生、数据大屏
  关闭登录：backend\.env 设 AUTH_ENABLED=false 后重启后端

【四、能耗实时对接（可选）】
  在 backend\.env 配置：
    ENERGY_API_URL=http://外部能耗接口
    ENERGY_SYNC_INTERVAL_SEC=300
  后端启动后自动拉取合并；平台运维页可「立即同步」。
  本地联调：python scripts\mock_energy_api.py → URL 填 http://127.0.0.1:9876/energy/latest

【五、其它】
  · 配置模板：backend\.env.example（LLM、百度语音、高德等）
  · Docker：docker compose up --build -d → http://localhost:8080
  · 小程序：miniprogram\（运维移动端，见 docs\USER_MANUAL.md） python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8765
  · API 契约：/docs 或 GET /api
  · 详细文档：docs\SOURCE_CODE_DOCUMENT.md、docs\USER_MANUAL.md、docs\API_FRONTEND.md

【排错】
  · {"detail":"Not Found"} → 确认后端已启动且端口与前端代理一致
  · WinError 10013 → 换端口，如 --port 18888
  · LLM 未配置 → 复制 .env.example 为 .env，填入 LLM_API_KEY 后重启
